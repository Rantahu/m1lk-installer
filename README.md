# M!lk
A Rice for your Debian Based machine. _Tested on Debian 10_

You can check out my dotefiles here at [https://github.com/S1MB10T3/dotfiles](https://github.com/S1MB10T3/dotfiles)


## Install Rice
Run this script to start the isntaller. 
```
sudo bash <(curl -s https://raw.githubusercontent.com/S1MB10T3/m1lk-installer/master/install.sh)
```
Currently the bash script needs to be run by sudo to install the packages needed for the script.
